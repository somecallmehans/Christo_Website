Reddit Background Switcher
About

I wrote this script because I like having a rotating desktop background, and I also like the albums covers at r/fakealbumcovers. The nice thing about this script is that almost any image based subreddit can be inserted in and it should work. Some of the challenges I had to work through were tailoring it specifically to macOS (since that's what I was developing on), and also how to handle images that weren't .jpg or .png. In the future I would like to rework this script for Windows, and also figure out a way to resize images so they don't get stretched.

Tech Used
  • Python3
  • Python Reddit API Wrapper (PRAW)
  • Pillow

Spotify Minute Tracker
About

I want to preface this by saying that last.fm does what this script is supposed to do, but like 1000x better. The idea is that you start the script and then begin a spotify session. The script detects that the session begins and starts a timer that starts and stops as the session starts and stops. Then later (I set it for early morning) it sends an email to the user with the number of minutes they listened the day prior. An expansion of this script would be to have it run year round and total the minutes for a whole year.

Tech Used
  • Python3
  • Spotipy
